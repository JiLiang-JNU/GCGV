# GCGV
This is the source code for the paper "GCGV: A Dual-Branch Hybrid Network Integrating Graph Attention, CNNs, and Vision Transformers for Enhanced Hyperspectral Image Classification," which is currently under review at The Multimeida Systems. Once the paper is published, we will make the code publicly available.

If you are a reviewer or researcher and need the code to verify the experimental results in our paper, please feel free to contact us via email: 6233114012@stu.jiangnan.edu.cn.

If our paper or code is helpful to your research, we would greatly appreciate it if you could cite our work. Thank you very much!
