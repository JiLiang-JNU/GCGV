# GCGV
This is the source code for the paper "GCGV: A Dual-Branch Hybrid Network Integrating Graph Attention, CNNs, and Vision Transformers for Enhanced Hyperspectral Image Classification," which is accepted at The Multimedia Systems. 

The link of our paper is: https://doi.org/10.1007/s00530-026-02300-2.
